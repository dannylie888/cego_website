# cego_website

Source repository for the **Cego Ceramics** company website.

## Repository Layout

```
cego_website/
├── cego_materials/     # Original product images & catalogue PDFs (source assets)
└── website/            # Next.js website — this is where you work
```

## Getting Started

All website code lives in the `website/` folder. Head there for the full setup guide:

```bash
cd website
```

Then follow the **[website/README.md](./website/README.md)** for installation and running instructions.

### TL;DR

```bash
cd website
npm install
npm run dev
# → http://localhost:3000
```
